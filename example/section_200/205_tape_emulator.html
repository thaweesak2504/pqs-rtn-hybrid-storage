    2. เอกสารออ้างอิง :
    1. Technical Manual Solid State Tape Emulator (SSTE)
  1. หน้าที่
    1. ระบบนี้ทําหน้าที่อะไร
      2. ๒๐๕.๑.๑ : ระบบนี้ทําหน้าที่อะไร
      เก็บโปรแกรมการทํางานของระบบ และข้อมูลการทํางาน
  2. ส่วนประกอบและชิ้นส่วนในส่วนประกอบของระบบ
    3. อ้างถึงเอกสารประกอบระบบ หรือตัวอุปกรณ์ เพื่อหาส่วนประกอบและชิ้นส่วนในส่วนประกอบ ดังต่อไปนี้ แล้วตอบคําถามที่กําหนด
    1. มีหน้าที่อะไร
    2. ใช้พลังงานหรือกําลังงานอะไรและได้รับมาจากไหน
    3. เมื่อขัดข้องหรือทํางานผิดปกติ มีอะไรเป็นสิ่งบอกเหตุ
    4. สีของไฟแสดงเริ่มต้นและเมื่อเปลี่ยนเป็นสีอะไร
    คำตอบ: ก. ข. ค. ง.
    1. SSTE Assembly"
      2. ๒๐๕.๒.๑ : SSTE Assembly"
      เก็บโปรแกรมการทํางานของระบบ เพื่อโหลดเข้าสู่ระบบ และเก็บข้อมูลสถานะ ของระบบต่างๆ ใช้ทดแทนการโหลดจากตลับเทปโปรแกรมเดิม
      ไฟฟ้าเรือ 115V.ac 60Hz Single Phase
      ไฟแสดงสถานะ "Error" จะติดสว่างสีแดง
    2. ไฟแสดงสถานะ "SSTE Status Indicator"
      1. ไฟแสดงสถานะ "Status Go"
      2. ไฟแสดงสถานะ "Error"
      3. ไฟแสดงสถานะ "Busy"
        4. ๒๐๕.๒.๒ : ไฟแสดงสถานะ "SSTE Status Indicator"
        แสดงสถานการทํางานของ SSTE
        ๒๐๕.๒.๒ ก. : ไฟแสดงสถานะ "Status Go"
        แสดงสถานการทํางานปกติ
        สีเขียว
        ๒๐๕.๒.๒ ข. : ไฟแสดงสถานะ "Error"
        แสดงการทํางานผิดปกติ มีข้อขัดข้องเกิดขึ้น
        สีแดง
        ๒๐๕.๒.๒ ค. : ไฟแสดงสถานะ "Busy"
        กระพริบเมื่อกําลังมีการถ่ายโอนข้อมูลระหว่าง SSTE กับระบบ CIWS
        กระพริบสีเขียว
    3. สวิตช์โหลดโปรแกรม "Program Load"
      4. ๒๐๕.๒.๓ : สวิตช์โหลดโปรแกรม "Program Load"
      กดสวิตช์เริ่มต้นโหลดโปรแกรมจาก SSTE
    4. ช่องเชื่อมต่อกับระบบ "System Interface Connector"
      5. ๒๐๕.๒.๔ : ช่องเชื่อมต่อกับระบบ "System Interface Connector"
      เชื่อมต่อสายสัญญาณข้อมูลระหว่าง SSTE กับระบบ CIWS
    5. ช่องเชื่อมต่อ RS-232 "RS -232 Interface Connector"
      6. ๒๐๕.๒.๕ : ช่องเชื่อมต่อ RS-232 "RS -232 Interface Connector"
      เชื่อมต่อกับคอมพิวเตอร์ภายนอกเพื่อโหลดโปรแกรมให้กับ SSTE
    6. แผ่นโปรแกรม CD
      7. ๒๐๕.๒.๖ : แผ่นโปรแกรม CD
      บรรจุโปรแกรมการทํางานของระบบ CIWS ที่ติดตั้งใน ทร. ไทย เท่านั้น
  3. หลักการทํางาน
    1. ส่วนประกอบต่างๆ ทํางานร่วมกันในระบบอย่างไร
      2. ๒๐๕.๓.๑ : ส่วนประกอบต่างๆ ทํางานร่วมกันในระบบอย่างไร
      ภายใน SSTE จะเก็บโปรแกรมการทํางานของระบบ CIWS ต้องโหลดโปรแกรมให้ ระบบ CIWS ก่อน
    2. เมื่อระบบขัดข้องหรือทํางานผิดปกติ มีอะไรเป็นสิ่งบอกเหตุ
      3. ๒๐๕.๓.๒ : เมื่อระบบขัดข้องหรือทํางานผิดปกติ มีอะไรเป็นสิ่งบอกเหตุ
      ไฟแสดงสถานะ "Error" จะติดสว่างสีแดง หรือไม่มีการแสดงของไฟแสดงต่างๆ เลย
  4. ค่าทํางานปกติ ค่าสูงสุด ต่ำสุด ของการทํางาน (ไม่ต้องอธิบาย)
  5. การเชื่อมต่อระบบ
    1. SSTE ทํางานเชื่อมต่อกับคอมพิวเตอร์ของระบบอย่างไร
      2. ๒๐๕.๕.๑ : SSTE ทํางานเชื่อมต่อกับคอมพิวเตอร์ของระบบอย่างไร
      ด้วยสายเชื่อมตอสัญญาณข้อมูล SSTE จะถ่ายโอนโปรแกรมที่บรรจุอยู่ภายในไปเก็บไว้ ในคอมพิวเตอร์ของระบบ CIWS
    2. SSTE เชื่อมต่อกับคอมพิวเตอร์ภายนอกได้อย่างไร
      3. ๒๐๕.๕.๒ : SSTE เชื่อมต่อกับคอมพิวเตอร์ภายนอกได้อย่างไร
      ด้วยสายเชื่อมต่อข้อมูลแบบ RS-232 ทําให้สามารถใช้คอมพิวเตอร์ทั่วไปที่มีหัวต่อแบบ นี้โหลดโปรแกรมจากแผ่น CD เข้าสู่ SSTE ได้
  6. ข้อระมัดระวังอันตราย
    1. มีข้อระมัดระวังอันตรายอะไรบ้างในการใช้งาน SSTE
      2. ๒๐๕.๖.๑ : มีข้อระมัดระวังอันตรายอะไรบ้างในการใช้งาน SSTE
      อุปกรณ์อิเล็กทรอนิกส์ภายในของ SSTE เป็นประเภทที่ไวต่อผลกระทบของไฟฟ้าสถิตย์ มาก (ElectroStatic Discharge Sensitive:ESDS) การจับต้องอุปกรณ์ดังกล่าวต้องทำอย่าง ระมัดระวังและเป็นไปตามมาตรฐาน MIL-STD-1686
