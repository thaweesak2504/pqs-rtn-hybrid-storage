1. ระบบเรดาร์ควบคุมการยิง Radar Weapon Assembly System
    2. เอกสารออ้างอิง :
    1. NAVSEA OP4154 CIWS Mk.15 Mods 1-6 (Phalanx) Vol.1 Pt.1,2
    2. NAVSEA OP4154 CIWS Mk.15 Mods 1-6 (Phalanx) Vol.2
    3. SW221-JO-MMO-010 thru 110, CIWS Mk.15 Mods 11-14
  1. หน้าที่
    1. ระบบนี้ทำหน้าที่อะไร
      2. ๒๐๑.๑.๑ :
      ชี้เป้าเพื่อที่จะทำการยิงและทำการยิง
  2. ส่วนประกอบและชิ้นส่วนในส่วนประกอบของระบบ
    3. อ้างถึงเอกสารประกอบระบบ หรือตัวอุปกรณ์ เพื่อหาส่วนประกอบและชิ้นส่วนในส่วนประกอบ ดังต่อไปนี้ แล้วตอบคำถามที่กำหนด
    1. มีหน้าที่อะไร
    2. ตำแหน่งที่ติดตั้งอยู่ที่ไหน
    3. อุปกรณ์นี้ใช้ป้องกันในลักษณะใด
    4. ในแต่ละตำแหน่งทำหน้าที่อะไร
    คำตอบ: ก. ข. ค. ง.
    1. ส่วนประกอบขับเคลื่อนทางหัน "Train Drive Platform Assembly"
      1. สวิตช์โยกตำแหน่งเก็บทางหัน "Mount Train Stow"
      2. สวิตช์โยกแท่นฐานปลอดภัย "Mount Safety"
      3. ขับเคลื่อนทางหันแบบแมนนวล "Mount Train Manual Drive"
      4. ขับเคลื่อนสลักเก็บตำแหน่งทางหัน "Train Stow Pin Manual Drive"
        5. ๒๐๑.๒.๑ : ส่วนประกอบขับเคลื่อนทางหัน "Train Drive Platform Assembly"
        ควบคุมตำแหน่งปืนทางหัน
        อยู่บนตู้ Barbette Assembly
        ๒๐๑.๒.๑ ก. : สวิตช์โยกตำแหน่งเก็บทางหัน "Mount Train Stow"
        ควบคุมการเคลื่อนตัวเข้าออกของ Stow Pin ทางหัน (Gearlock) และมีไฟแสดง เมื่อ Switch ตัวนี้ทำงานในตำแหน่ง Engage และ Retract.
        ๒๐๑.๒.๑ ข. : สวิตช์โยกแท่นฐานปลอดภัย "Mount Safety"
        ห้ามหันและกระดกปืน
        ในแต่ละตำแหน่งทำหน้าที่ดังนี้
        1. OPERATE เป็นตำแหน่งใช้งานปกติของระบบ
        2. SAFE ห้ามหัน/กระดกและตัดวงจรไฟยิง
        3. RELOAD เหมือนกับตำแหน่ง SAFE และทำให้ระบบ Hydraulic ทำงานขณะทำการ Loading และ Downloading ลูกปืน
        ๒๐๑.๒.๑ ค. : ขับเคลื่อนทางหันแบบแมนนวล "Mount Train Manual Drive"
        เป็นที่หันปืนโดยใช้พวงหมุนขนาด 0.5 นิ้ว และปรับแต่ง Stow Pin ให้อยู่ในตำแหน่ง Engage พอดี
        ๒๐๑.๒.๑ ง. : ขับเคลื่อนสลักเก็บตำแหน่งทางหัน "Train Stow Pin Manual Drive"
        เพื่อที่จะหมุน TRAIN Stow Pin ให้อยู่ในตำแหน่ง Retract หรือ Engage ด้วยมือ
    2. ส่วนประกอบขับเคลื่อนทางกระดก "Elevation Drive Assembly"
      1. ขับเคลื่อนสลักเก็บตำแหน่งทางกระดก "Elevation Stow Pin Manual Drive"
      2. สวิตช์โยกตำแหน่งเก็บทางกระดก "Elevation Stow Pin"
      3. คันมือหมุนทางกระดก "Elevation Manual Hand Crank"
        4. ๒๐๑.๒.๒ : ส่วนประกอบขับเคลื่อนทางกระดก "Elevation Drive Assembly"
        ควบคุมตำแหน่งปืนทางกระดก
        อยู่บน Train Platform Assembly
        ๒๐๑.๒.๒ ก. : ขับเคลื่อนสลักเก็บตำแหน่งทางกระดก "Elevation Stow Pin Manual Drive"
        เพื่อที่จะหมุนให้ Elevation Stow Pin อยู่ในตำแหน่ง Stow Engage หรือ Retract ด้วยมือ
        ๒๐๑.๒.๒ ข. : สวิตช์โยกตำแหน่งเก็บทางกระดก "Elevation Stow Pin"
        ควบคุม Elevation Stow Pin ให้อยู่ในตำแหน่ง Stow หรือ Retract
        ๒๐๑.๒.๒ ค. : คันมือหมุนทางกระดก "Elevation Manual Hand Crank"
        เป็นเครื่องมือใช้สำหรับปลดเบรคและหมุนกระดกปืน ขึ้น-ลง
    3. ปืน 20 mm.
      1. ไฮดรอลิกส์ขับเคลื่อนปืน "Gun Hydraulic Drive"
      2. ขอเกียวลำกล้องเกลี้ยง "Clearing Sector Hold Back Tool"
        3. ๒๐๑.๒.๓ : ปืน 20 mm.
        ทำหน้าที่ยิงทำลายเป้า
        ติดตั้งอยู่บน Elevation Yoke Structure
        ๒๐๑.๒.๓ ก. : ไฮดรอลิกส์ขับเคลื่อนปืน "Gun Hydraulic Drive"
        ขับเคลื่อนปืนและระบบการลำเลียงด้วยระบบไฮโดรลิกในขณะ Uploading/Downloading, Firing
        ๒๐๑.๒.๓ ข. : ขอเกียวลำกล้องเกลี้ยง "Clearing Sector Hold Back Tool"
        เมื่อติดตั้งจะรักษาลูกเลื่อน (Breech Bolt) ให้อยู่ในทางเดิน Clearing Cam Path
        ป้องกันไม่ให้ลูกปืนเข้าไปในรังเพลิง
    4. ระบบลำเลียงและบรรจุ "Ammunition Handling/Conveyor System"
      1. Feed Chute
      2. Element Chuting
      3. Entrance Unit
      4. Exit Unit
      5. Exit Unit Drive Socket
      6. Loading Gate
      7. Ammunition Drum
      8. Drum Timing Pin
      9. Hydraulic/Pneumatic Control Actuator
        10. ๒๐๑.๒.๔ : ระบบลำเลียงและบรรจุ "Ammunition Handling/Conveyor System"
        รับ - ส่ง ลูกปืนจาก DRUM ถึงตัวปืน และจากตัวปืนถึง DRUM
        ใต้ตัวปืน ๒๐ มม
        ๒๐๑.๒.๔ ก. : Feed Chute
        มีหน้าที่ใช้เป็นทางลำเลียงลูกปืน
        ๒๐๑.๒.๔ ข. : Element Chuting
        มีหน้าที่ส่ง Element ระหว่าง DRUM และปืน
        ๒๐๑.๒.๔ ค. : Entrance Unit
        ส่งลูกปืนจาก Chuting เข้าสู่ Drum
        ๒๐๑.๒.๔ ง. : Exit Unit
        มีหน้าที่ส่งลูกปืนจาก Drum ไปยัง Feed Chute
        ๒๐๑.๒.๔ จ. : Exit Unit Drive Socket
        เป็นที่ใส่พวงหมุน (Handcrank) เพื่อทำการ Cycle ลูกปืนด้วยมือ
        ๒๐๑.๒.๔ ฉ. : Loading Gate
        เป็นส่วนประกอบของ Exit Unit เมื่อเปิดโดยการยกขึ้นลูกปืนจะสามารถผ่านได้ ขณะ Upload และ Download
        ๒๐๑.๒.๔ ช. : Ammunition Drum
        มีหน้าที่เป็นที่เก็บขณะ Loading และส่งผ่านลูกปืน
        ๒๐๑.๒.๔ ซ. : Drum Timing Pin
        มีหน้าที่ใช้งานสำหรับตั้ง Timing ของ Drum
        ๒๐๑.๒.๔ ฌ. : Hydraulic/Pneumatic Control Actuator
        เป็นคันโยกเพื่อควบคุมการหมุนของตัวปืนและส่วนลำเลียงลูกปืนโดยระบบ ไฮโดรลิก
    5. ส่วนประกอบตัวบรรจุลูกปืน "Ammunition Loader Assembly"
      1. Loader Timing Pin
      2. Link Chute
      3. Loader Gears (Rs and LS)
        4. ๒๐๑.๒.๕ : ส่วนประกอบตัวบรรจุลูกปืน "Ammunition Loader Assembly"
        มีหน้าที่ใช้งานสำหรับการ Upload และ Download ลูกปืนเท่านั้น
        ติดตั้งบน Exit Unit
        ๒๐๑.๒.๕ ก. : Loader Timing Pin
        มีหน้าที่ตั้ง Timing ของ Loader
        ๒๐๑.๒.๕ ข. : Link Chute
        ใช้ขณะ Download เพื่อนำลูกปืนที่ยิงแล้วออกจากระบบ
        ๒๐๑.๒.๕ ค. : Loader Gears (Rs and LS)
        หน้าที่ทำให้ Gear Timing ในขณะ Upload และ Download เป็นไปอย่างถูกต้อง
  3. หลักการทำงาน
    1. ส่วนประกอบต่างๆ ทำงานร่วมกันในระบบอย่างไร
      2. ๒๐๑.๓.๑ : ส่วนประกอบต่างๆ ทำงานร่วมกันในระบบอย่างไร
      มีเพื่อชี้ตำแหน่งเป้าหมายและบรรจุลูกปืน
    2. เมื่อระบบขัดข้องหรือทำงานผิดปกติ มีอะไรเป็นสิ่งบอกเหตุ
      3. ๒๐๑.๓.๒ : เมื่อระบบขัดข้องหรือทำงานผิดปกติ มีอะไรเป็นสิ่งบอกเหตุ
      ไม่สามารถทำการยิงได้
  4. ค่าทำงานปกติ ค่าสูงสุด ต่ำสุด ของการทำงาน (ไม่ต้องอธิบาย)
  5. การเชื่อมต่อระบบ
    1. ถ้าขาดสิ่งดังต่อไปนี้จะมีผลกระทบต่อระบบอย่างไร
      1. ขาดการควบคุมจาก LCP และ/หรือ RCP
      2. ไม่มีไฟฟ้าเรือจ่ายให้กับระบบ
        3. ๒๐๑.๕.๑ ก. : ขาดการควบคุมจาก LCP และ/หรือ RCP
        จะไม่ตอบสนองต่อสัญญาณคำสั่งต่าง ๆ
        ๒๐๑.๕.๑ ข. : ไม่มีไฟฟ้าเรือจ่ายให้กับระบบ
        ไม่สามารถทำการยิงได้
  6. ข้อระมัดระวังอันตราย
    1. มีข้อระมัดระวังอันตรายอะไรบ้าง ในการปฏิ บัติงาน ดังต่อไปนี้
      1. ขณะกระดกปืน โดยวิธีใช้คันมือหมุน
      2. ขณะบรรจุและถอนบรรจุลูกปืน
        3. ๒๐๑.๖.๑ ก. : ขณะกระดกปืน โดยวิธีใช้คันมือหมุน
        ต้องแน่ใจว่าได้จับพวงหมุนให้แน่น ขณะใส่พวงหมุนเพื่อหมุนปืน
        ๒๐๑.๖.๑ ข. : ขณะบรรจุและถอนบรรจุลูกปืน
        มีข้อระมัดระวังอันตราย ดังนี้
        1. ต้องแน่ใจว่าโหมดการทำงานของเครื่องอยู่ที่ Standby และงดแพร่คลื่นเรดาร์ทุกชนิด
        2. สำหรับลูกปืนที่ใช้งานไม่ได้ควรทิ้งทะเล
        3. ประกาศแจ้งให้ทราบทั่วกันถึงอันตรายที่เกิดจากการแพร่คลื่นต่อระบบอาวุธ (Hero) ก่อนการขนย้ายลูกปืน
        4. บริเวณรอบๆ แท่นปืน ห้ามเข้านอกบริเวณก่อน 30 นาที หลังจากยิงเสร็จ
        5. ต้องสวมถุงมืออย่างหนาขณะทำงาน
        6. หลีกเลี่ยงการสัมผัส Primer และ Primer สัมผัสกับวัตถุที่เป็นโลหะ
        7. พื้นที่ปฏิบัติงานต้องเรียบร้อย
        8. ต้องระมัดระวังมือและเสื้อผ้าที่จะโดนเกี่ยวขณะทำงาน
        9. ต้องไม่เดินระบบไฮโดรลิกนานติดต่อเกิน ๗ นาที
        10. ต้องติดตั้ง Sector Holdback Tool ให้ถูกต้อง
        11. ต้องถอด Mount Safety Connector ที่ Lcp ออก
        12. ตรวจสอบ Guide Bolt ก่อนทำการ Upload และ Download
