        บนแผงควบคุม RCP Power Panel
        ๒๐๓.๒.๑๓ ก. : สวิตช์ "Primary Color/Off/Alternate Color"
        ประยุกต์ไฟเพื่อทดสอบหลอดไฟและตัวเลขแสดงผลทั้งหมดโดยทดสอบ Primary หรือ Alternate Color
        เพื่อประยุกต์ไฟเพื่อที่จะทดสอบหลอดไฟในแต่ละตำแหน่ง
        ๒๐๓.๒.๑๓ ข. : สวิตช์ปุ่มกด "RCP"
        ใส่คำตอบที่นี่
        เลือก RCP เพื่อที่จะทําการทดสอบหลอดไฟ
    14. ภาคส่วนควบคุมการเดินระบบ "Power Control Section"

      2. สวิตช์ "Power On"
      3. ปุ่มกด "CB1"
        4. ๒๐๓.๒.๑๔ : ภาคส่วนควบคุมการเดินระบบ "Power Control Section"

        ๒๐๓.๒.๑๔ ก. : สวิตช์ "Power Off (avail)"
        ลิกไฟให้กับ RCP ไฟแสดงสีเหลืองติดแสดงว่า Power พร้อมที่จะใช้งาน
        ๒๐๓.๒.๑๔ ข. : สวิตช์ "Power On"
        เมื่อกดปุ่มนี้ประยุกต์ไฟให้กับ RCP ไฟติดสีขาวแสดงว่าได้ประยุกต์ไฟให้กับ RCP แล้ว
        ขาว / -
        ๒๐๓.๒.๑๔ ค. : ปุ่มกด "CB1"
        ในตําแหน่งกดลงแสดงว่าได้จ่ายไฟ 115V. 60Hz ไปยัง Power Off (avail) Switch และ Power On Switch
        มาจากแผงสวิทช์บอร์ดเรือ
        ป้องกันการ Overload

  3. หลักการทํางาน
    1. ส่วนประกอบต่างๆ ทํางานร่วมกันในระบบอย่างไร
      2. ๒๐๓.๓.๑ : ส่วนประกอบต่างๆ ทํางานร่วมกันในระบบอย่างไร
      RCP จะควบคุมและแสดงผลของระบบ CIWS โดยควบคุมจากห้องศูนย์ยุทธการ (CIC)
    2. เมื่อระบบขัดข้องหรือทํางานผิดปกติ มีอะไรเป็นสิ่งบอกเหตุ
      3. ๒๐๓.๓.๒ : ส่วนประกอบต่างๆ ทํางานร่วมกันในระบบอย่างไร
      ไม่สามารถควบคุมโหมดในการทํางานและการแสดงผล
  4. ค่าทํางานปกติ ค่าสูงสุด ต่ำสุด ของการทํางาน (ไม่ต้องอธิบาย)
  5. การเชื่อมต่อระบบ
    1. ถ้าขาดสิ่งดังต่อไปนี้จะมีผลกระทบต่อระบบอย่างไร
      1. ไม่มีไฟฟ้าเรือจ่ายให้กับระบบ
      2. การส่งถ่ายข้อมูลภายใน Remote Control Panel (RCP) ไม่ได้
        3. ๒๐๓.๕.๑ ก. : ไม่มีไฟฟ้าเรือจ่ายให้กับระบบ
        ไม่สามารถเดินระบบ RCP ได้
        ๒๐๓.๕.๑ ข. : การส่งถ่ายข้อมูลภายใน Remote Control Panel (RCP) ไม่ได้
        ไม่สามารถเดินระบบ RCP ได้
  6. ข้อระมัดระวังอันตราย
    1. มีข้อระมัดระวังอันตรายอะไรบ้าง ในการปฏิ บัติงาน ดังต่อไปนี้
      1. การเลือกโหมดการทํางาน AIR READY
      2. การเลือกขั้นการทํางาน AAW AUTO
      3. การเปลี่ยนสวิทช์ PRE ARM/ARM/SAFE ไว้ตําแหน่ง ARM
        4. ๒๐๓.๖.๑ ก. : การเลือกโหมดการทำงาน AIR READY
        ต้องแน่ใจว่าบริเวณรอบๆ แท่นปืนต้องปราศจากผู้คนก่อนเลือกโหมดนี้ ต้องกด Mount Warning ประมาณ ๕ วินาที
        ๒๐๓.๖.๑ ข. : การเลือกขั้นการทำงาน AAW AUTO
        เมื่อเลือกโหมดนี้ต้องแน่ใจว่าต้องไม่มีอากาศยานหรือเรือฝ่ายเดียวกันอยู่อย่างน้อย ๖ ไมล์ในมุมที่ทำการต่อตีเป้า เพราะถ้าเลือกโหมดนี้ระบบ CIWS จะทำงานเต็มระบบตั้งแต่ค้นหา ตรวจจับ ติดตามต่อตีเป้า และทำการยิงโดยอัตโนมัติ
        ๒๐๓.๖.๑ ค. : การเปลี่ยนสวิตช์ PRE ARM/ARM/SAFE ไว้ตำแหน่ง ARM
        เมื่อสวิตช์อยู่ในตำแหน่ง ARM จะมีการต่อวงจรไฟยิง การยิงอาจเกิดขึ้นได้ถ้าเงื่อนไขการยิงอื่นๆ พร้อม ต้องไม่เปลี่ยนสวิตช์ให้อยู่ในตำแหน่ง ARM นอกจากทำการยิง หรือได้รับอนุญาตจากผู้มีอำนาจหน้าที่เท่านั้น
