    12. ภาคส่วนควบคุมการยิง "Fire Control Section"
      1. สวิตช์ปุ่มกดเลือกยิงชุด "Burst 60/Burst 100"

      3. สวิตช์ปุ่มกดหยุดติดตามเป้า "Break Engage"
      4. สวิตช์ปุ่มกดการขึ้นนก "PRE ARM/ARM/SAFE"
      5. สวิตช์ปุ่มกดหยุดยิง "Hold Fire"
      6. สวิตช์ปุ่มกดยิง "Fire"
        7. ๒๐๒.๒.๑๒ : ภาคส่วนควบคุมการยิง "Fire Control Section"
        ควบคุมเงื่อนไขในการยิง
        มุมขวาล่างของตู้ LCP
        ๒๐๒.๒.๑๒ ก. : สวิตช์ปุ่มกดเลือกยิงชุด "Burst 60/Burst 100"
        เป็นปุ่มเลือกว่าจะยิงชุด 100 หรือ 60 นัด ขณะทําการยิง PAC Fire Mode

        แสดงลูกปืนที่เหลืออยู่ในระบบบรรจุ
        ๒๐๒.๒.๑๒ ค. : สวิตช์ปุ่มกดหยุดติดตามเป้า "Break Engage"
        เมื่อกดปุ่มนี้ไฟจะติดสีเหลืองเมื่อ WCG คอมพิวเตอร์รับทราบว่ามีการขัดจังหวะโปรแกรมการทํางาน
        เหลือง / -

        มีหน้าที่ใช้สำหรับ Mode AIR Ready หรือสูงกว่า เป็นการเลือกสถานะของปืนว่า SAFE หรือ ARM ไฟ PRE ARM และ ARM ติดสีแดงแสดงว่าเป็นการต่อวงจรไฟยิง ไฟ SAFE สีเขียวติดแสดงว่าเป็นการไม่ต่อวงจรไฟยิง

        แดง / ไม่เขียว / -

        เป็นปุ่มที่มีฝาครอบ

  3. หลักการทํางาน
    1. ส่วนประกอบต่างๆ ทํางานร่วมกันในระบบอย่างไร
      2. ๒๐๒.๓.๑ : ส่วนประกอบต่างๆ ทํางานร่วมกันในระบบอย่างไร
      LCP จะยอมให้มีการป้อนข้อมูลเพื่อตีเป้าไปใน WCG คอมพิวเตอร์ ถ้า RCP ไม่สามารถเดินเครื่องได้ LCP สามารถเป็นแผงควบคุมหลักของระบบได้นอกจากนั้น LCP ยังมีหน้าที่เกี่ยวกับการซ่อมบำรุงรักษาอีกด้วย
    2. เมื่อระบบขัดข้องหรือทํางานผิดปกติ มีอะไรเป็นสิ่งบอกเหตุ
      3. ๒๐๒.๓.๒ : เมื่อระบบขัดข้องหรือทํางานผิดปกติ มีอะไรเป็นสิ่งบอกเหตุ
      มีสิ่งบอกเหตุ ดังนี้
      1. ไม่สามารถที่จะเริ่มต้นการทดสอบพิเศษได้
      2. ไม่สามารถป้อนข้อมูลทดสอบและข้อมูลต่อตีเป้าลงในคอมพิวเตอร์ได้
      3. ไม่สามารถเรียกค่าจุดทดสอบและข้อมูลระบบที่เสียออกมาดูได้
      4. ไม่สามารถแสดงผลสถานะและข้อมูลของระบบได้
      5. ไม่สามารถส่งข้อมูลทิศหัวเรือให้คอมพิวเตอร์ได้
      6. ไม่สามารถควบคุม Mode การทํางานต่างๆ ของระบบ CIWS ได้
  4. ค่าทํางานปกติ ค่าสูงสุด ต่ำสุด ของการทํางาน (ไม่ต้องอธิบาย)
  5. การเชื่อมต่อระบบ
    1. ถ้าขาดสิ่งดังต่อไปนี้จะมีผลกระทบต่อระบบอย่างไร
      1. ไม่มีไฟฟ้าเรือจ่ายให้กับระบบ
      2. การส่งถ่ายข้อมูลภายใน Local Control Panel (LCP) ไม่ได้
        3. ๒๐๒.๕.๑ ก. : ไม่มีไฟฟ้าเรือจ่ายให้กับระบบ
        LCP ไม่สามารถควบคุมได้
        ๒๐๒.๕.๑ ข. : การส่งถ่ายข้อมูลภายใน Local Control Panel (LCP) ไม่ได้
        LCP ไม่สามารถควบคุมได้
    2. LCP ทํางานร่วมกับคอมพิวเตอร์ WCG อย่างไร
      3. ๒๐๒.๕.๒ : LCP ทํางานร่วมกับคอมพิวเตอร์ WCG อย่างไร
      โดยการรับ/ส่งข้อมูลระหว่างกันเพื่อที่จะควบคุมระบบโดย LCP
  6. ข้อระมัดระวังอันตราย
    1. มีข้อระมัดระวังอันตรายอะไรบ้าง ในการปฏิ บัติงาน ดังต่อไปนี้
      1. การเลือกโหมดการทํางาน AIR READY
      2. การเลือกควบคุมการหมุนแท่นยิงแบบ LOCAL
      3. การเลือกขั้นการทํางาน AAW AUTO
      4. การเปลี่ยนสวิทช์ PRE ARM/ARM/SAFE ไว้ตําแหน่ง ARM
        5. ๒๐๒.๖.๑ ก. : การเลือกโหมดการทํางาน AIR READY
        เมื่อเลือกโหมดนี้ปืนจะหมุนไปในตําแหน่ง AIR Ready ต้องแน่ใจว่ารอบๆ แท่นปืน ปราศจากผู้คนและต้องกดปุ่มเตือน Mount Warning ก่อนกดปุ่มนี้ ควรกดค้างไว้ประมาณ ๕ วินาที
        ๒๐๒.๖.๑ ข. : การเลือกควบคุมการหมุนแท่นยิงแบบ LOCAL
        เมื่อเลือกโหมดการทํางานนี้จะทําให้ปืนหันและกระดกไปตามค่าจากหน้าปัด TR และ EL Synchro Dials ก่อนเลือกโหมดนี้ต้องปรับ Synchro Dials ไปที่ตำแหน่งมุมที่ ต้องการและกดสัญญาณเตือนการหมุนของแท่นปืน Mount Warning อย่างน้อย ๕ วินาที
        ๒๐๒.๖.๑ ค. : การเลือกขั้นการทํางาน AAW AUTO
        ก่อนเลือกโหมดการทํางานนี้ต้องแน่ใจว่าไม่มีอากาศยานหรือเรือฝ่ายเดียวกันอยู่ใน ระยะ ๖ ไมล์ในแบริ่งที่จะทําการต่อตีเป้าเพราะถ้าเลือกโหมดนี้ CIWS จะทํางานตั้งแต่ค้นหา ติดตามและทําลายเป้าภายใต้เงื่อนไข โดยไม่มีสัญญาณเตือน
        ๒๐๒.๖.๑ ง. : การเปลี่ยนสวิทช์ PRE ARM/ARM/SAFE ไว้ตำแหน่ง ARM
        เมื่อสวิตช์อยู่ในตำแหน่ง ARM จะมีการต่อวงจรไฟยิงต้องไม่เปลี่ยนสวิตช์ให้อยู่ใน ตำแหน่ง ARM นอกเสียจากว่าจะทำการยิงและได้รับอนุญาตจากผู้มีอำนาจหน้าที่
