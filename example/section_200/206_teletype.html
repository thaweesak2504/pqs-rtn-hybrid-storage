1. ระบบเครื่องพิมพ์ Teletype (KSR 43) System
    2. เอกสารออ้างอิง :
    1. NAVSEA OP4154 Close In Weapon System Mk.15 Mods 1-6 (Phalanx) Vol.2
    2. TTeletype Corporation How to Operate Manual 367
  1. หน้าที่
    1. ระบบนี้ทําหน้าที่อะไร
      2. ๒๐๖.๑.๑ : ระบบนี้ทําหน้าที่อะไร
      เพื่อติดต่อกับคอมพิวเตอร์โดยการป้อนข้อมูลเข้าและแสดงผลข้อมูลโดยการพิมพ์ข้อมูลออกมาเพื่อดูข้อมูล
  2. ส่วนประกอบและชิ้นส่วนในส่วนประกอบของระบบ
    3. อ้างถึงเอกสารประกอบระบบ หรือตัวอุปกรณ์ เพื่อหาส่วนประกอบและชิ้นส่วนในส่วนประกอบ ดังต่อไปนี้ แล้วตอบคําถามที่กําหนด
    1. มีหน้าที่อะไร
    2. ตําแหน่งที่ติดตั้งอยู่ที่ไหน
    3. ตําแหน่งที่ใช้งานปกติอยู่ที่ตําแหน่งใด
    คำตอบ: ก. ข. ค.
    1. ส่วนควบคุมและแสดงผล "Control and Status"
      1. Local
      2. DATA
      3. TERM READY
      4. INTRPT
      5. ALARM
      6. PRINTER TEST
      7. PARITY
      8. Duplex
      9. CPS (Character Per Second)
        10. ๒๐๖.๒.๑ : ส่วนควบคุมและแสดงผล "Control and Status"
        เพื่อป้อนข้อมูลติดต่อสื่อสาร/โหมดในการทํางาน และ การทดสอบการซอม บํารุงรักษา
        ด้านหน้าของ Teletype
        ๒๐๖.๒.๑ ก. : Local
        ยอมให้มีการป้อนข้อมูลแบบ Local
        แผงด้านบนซ้าย
        ไฟไม่ติด
        ๒๐๖.๒.๑ ข. : DATA
        ยอมให้มีการป้อนข้อมูล/และแสดงผลข้อมูล
        แผงด้านบนซ้าย
        ปกติไฟติด
        ๒๐๖.๒.๑ ค. : TERM READY
        แสดงสถานะ Terminal Operating Status
        แผงด้านบนซ้าย
        ไฟไม่ติด
        ๒๐๖.๒.๑ ง. : INTRPT
        ขัดจังหวะการส่งข้อมูล
        แผงด้านบนซ้าย
        ไฟไม่ติดสว่าง
        ๒๐๖.๒.๑ จ. : ALARM
        ไฟติดเมื่อมีเงื่อนไข Alarm
        แผงด้านบนซ้าย
        ไฟไม่ติด
        ๒๐๖.๒.๑ ฉ. : PRITER TEST
        ยอมให้มีการทดสอบ Teletype โดยอิสระ
        แผงด้านบนขวา
        ไฟไม่ติดสว่าง
        ๒๐๖.๒.๑ ช. : PARITY
        ยอมให้มีการทดสอบ Teletype
        แผงด้านบนขวา
        ปกติกดขึ้น
        ๒๐๖.๒.๑ ซ. : Duplex
        ควบคุมโหมดในการรับ-ส่งข้อมูล Teletype
        แผงด้านบนขวา
        ปกติกดลง
        ๒๐๖.๒.๑ ฌ. : CPS (Character Per Second)
        เพื่อร่วมจังหวะข้อมูลระหว่างตัวส่งตัวรับเป็นไปอย่างถูกต้อง
        แผงด้านบนขวา
        กดลง
    2. สวิตช์ "ON/OFF"
      3. ๒๐๖.๒.๒ : สวิตช์ "ON/OFF"
      จ่ายและเลิกไฟ AC ให้กับ Teletype
      มุมด้านล่างซ้าย (ด้านหลัง)
      ตําแหน่ง ON
    3. ปุ่มแป้นพิมพ์ "Keyboard"
      4. ๒๐๖.๒.๓ : ปุ่มแป้นพิมพ์ "Keyboard"
      ยอมให้มีการป้อนข้อมูล
      อยู่ใต้ Operational Controls and Status Indicators
    4. ตลับผ้าหมึก "Ribbon Cartridge"
      5. ๒๐๖.๒.๔ : ตลับผ้าหมึก "Ribbon Cartridge"
      ทําให้การพิมพ์ข้อมูลออกมาชัดเจน
      อยู่ภายใน Teletype
  3. หลักการทํางาน
    1. ส่วนประกอบต่างๆ ทํางานร่วมกันในระบบอย่างไร
      2. ๒๐๖.๓.๑ : ส่วนประกอบต่างๆ ทํางานร่วมกันในระบบอย่างไร
      ป้อนข้อมูล - พิมพ์ข้อมูลแสดงผล
    2. เมื่อระบบขัดข้องหรือทํางานผิดปกติ มีอะไรเป็นสิ่งบอกเหตุ
      3. ๒๐๖.๓.๒ : เมื่อระบบขัดข้องหรือทํางานผิดปกติ มีอะไรเป็นสิ่งบอกเหตุ
      Printer ไม่สามารถทํางานในโหมด Local หรือ Remote ได้
  4. ค่าทํางานปกติ ค่าสูงสุด ต่ำสุด ของการทํางาน (ไม่ต้องอธิบาย)
  5. การเชื่อมต่อระบบ
    1. ถ้าไม่มีไฟฟ้ากําลังจ่ายให้กับ Teletype จะมีผลกระทบต่อระบบอย่างไร
      2. ๒๐๖.๕.๑ : ถ้าไม่มีไฟฟ้ากําลังจ่ายให้กับ Teletype จะมีผลกระทบต่อระบบอย่างไร
      Printer ไม่สามารถทํางานในโหมด Local หรือ Remote ได้
    2. Teletype ทํางานร่วมกับ Tape Control Mk. 179 อย่างไร
      3. ๒๐๖.๕.๒ : Teletype ทํางานร่วมกับ Tape Control Mk. 179 อย่างไร
      โดยการรับ - ส่ง ข้อมูลซึ่งกันและกัน
  6. ข้อระมัดระวังอันตราย
    1. มีข้อระมัดระวังอันตรายอย่างไรในการเปลี่ยนตลับ Ribbon
      2. ๒๐๖.๖.๑ : ระบบนี้ทําหน้าที่อะไร
      เพื่อติดต่อกับคอมพิวเตอร์โดยการป้อนข้อมูลเข้าและแสดงผลข้อมูลโดยการพิมพ์ข้อมูลออกมาเพื่อดูข้อมูล
