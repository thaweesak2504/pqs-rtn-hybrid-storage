1. ระบบอิเล็กทรอนิคส์ Electronics (ELX) Enclosure System
    2. เอกสารออ้างอิง :
    1. NAVSEA OP4154 CIWS Mk.15 Mods 1-6 (Phalanx) Vol.1 Pt.1,2
    2. NAVSEA OP4154 CIWS Mk.15 Mods 1-6 (Phalanx) Vol.2
    3. SW221-JO-MMO-010 thru 110, CIWS Mk.15 Mods 11-14
  1. หน้าที่
    1. ระบบนี้ทําหน้าที่อะไร
      2. ๒๐๔.๑.๑ : ระบบนี้ทําหน้าที่อะไร
      เป็นที่ติดตั้งของระบบอิเล็กทรอนิกส์ มีการป้องกันมิดชิดและมีระบบรักษาอุณภูมิ
  2. ส่วนประกอบและชิ้นส่วนในส่วนประกอบของระบบ
    3. อ้างถึงเอกสารประกอบระบบ หรือตัวอุปกรณ์ เพื่อหาส่วนประกอบและชิ้นส่วนในส่วนประกอบ ดังต่อไปนี้ แล้วตอบคําถามที่กําหนด
    1. มีหน้าที่อะไร
    2. ตําแหน่งที่ติดตั้งอยู่ที่ไหน
    3. ใช้พลังงานหรือกําลังงานอะไรและได้รับมาจากไหน
    4. แบบการทํางานหรือการควบคุมมีอะไรบ้าง
    5. อุปกรณ์ป้องกันหรือให้ความปลอดภัยมีอะไรบ้าง
    6. อุปกรณ์นี้ใช้ป้องกันในลักษณะใด
    7. ในแต่ละตําแหน่งทําหน้าที่อะไร
    8. สีของไฟแสดงเริ่มต้นและเมื่อเปลี่ยนเป็นสีอะไร
    คำตอบ: ก. ข. ค. ง. จ. ฉ. ช. ซ.
    1. (2A1) Elevation Servo Electronics Removable Unit
      1. สวิตช์ "Reset/Norm/Disable"
      2. ไฟแสดง "Status Go"
        3. ๒๐๔.๒.๑ : (2A1) Elevation Servo Electronics Removable Unit
        ควบคุมแท่นปืนทางกระดก
        มุมบนด้านซ้ายของตู ELX (2A1)
        ๒๐๔.๒.๑ ก. : สวิตช์ "Reset/Norm/Disable"
        ในแต่ละตำแหน่งมีหน้าที่ ดังนี้
        1. Reset : เพื่อรีเซ็ต Elevation Status GO หลังจากมีการขัดจังหวะ
        2. Norm : เป็นตำแหน่งการทำงานปกติ
        3. Disable : เป็นการขัดจังหวะวงจร Mount Elevation Drive Circuit ส่วนของ วงจรอื่นสามารถทดสอบได้ตามปกติ
        มุมล่างซ้ายของ 2A1
        เป็น Toggle Switch แบบ ๓ ตำแหน่ง
        ๒๐๔.๒.๑ ข. : ไฟแสดง "Status Go"
        ไฟติดสีเขียวเมื่อสถานะปืนทางกระดกพร้อมและสามารถกระดกปืน ขึ้นลงได้
        มุมบนด้านขวา
    2. (2A4) Train Servo Electronic Removable Unit
      1. สวิตช์ "Reset/Norm/Disable"
      2. ไฟแสดง "Status Go"
        3. ๒๐๔.๒.๒ : (2A4) Train Servo Electronic Removable Unit
        ควบคุมแท่นปืนทางหัน
        อยู่ใต้ Elevation Servo Electronic Removable Unit
        ๒๐๔.๒.๒ ก. : สวิตช์ "Reset/Norm/Disable"
        ในแต่ละตำแหน่งมีหน้าที่ ดังนี้
        1. Reset : เพื่อรีเซ็ต Elevation Status GO หลังจากมีการขัดจังหวะ
        2. Norm : เป็นตำแหน่งการทำงานปกติ
        3. Disable : เป็นการขัดจังหวะวงจร Mount Elevation Drive Circuit ส่วนของ วงจรอื่นสามารถทดสอบได้ตามปกติ
        อยู่ที่มุมล่างด้านซ้าย
        เป็นสวิทช์แบบ Toggle ๓ ทาง
        ๒๐๔.๒.๒ ข. : ไฟแสดง "Status Go"
        แสดงสถานะเพื่อบอกว่าสถานะของแท่นปืนพร้อมที่จะหันซ้าย - ขวาได้
        อยู่ด้านบนขวาของตู้อุปกรณ์ 2A4
    3. (2A5) Gun Control Unit (GCU) Removable Unit
      1. ไฟแสดง "System Status Go"
      2. สวิตช์กุญแจ "Fire Intergrity"
      3. สวิตช์ "Gun Maintenance Clear Command"
        4. ๒๐๔.๒.๓ : (2A5) Gun Control Unit (GCU) Removable Unit
        ควบคุมการจ่ายไฟและคําสั่งต่างๆ ไปยังปืนพร้อมทั้งบอกสถานะของปืน
        อยู่ด้านล่างซ้ายของตู้อุปกรณ์ ELX
        ๒๐๔.๒.๓ ก. : ไฟแสดง "System Status Go"
        ไฟติดสีเขียวเมื่อสถานะของปืนพร้อม (GO)
        อยู่ด้านบนสุดซ้าย
        ๒๐๔.๒.๓ ข. : สวิตช์กุญแจ "Fire Intergrity"
        ในตําแหน่ง Open จะตัดไฟยิงไปยังปืน ในตําแหน่ง Close จะต่อไฟยิงให้กับปืน ปกติต้องอยู่ในตําแหน่ง Open จะเปลี่ยนเป็น Close ก่อนทําการยิงเท่านั้น
        ด้านบนสุดด้านขวาของ 2A5
        ๒๐๔.๒.๓ ค. : สวิตช์ "Gun Maintenance Clear Command"
        จะกดก็ต่อเมื่ออยู่ในโหมด AIR Ready หรือสูงกว่าจะเป็นการ Disable ไฟยิง แต่ปืนจะมีอาการ Cycle เหมือนทําการยิงทุกประการ
        อยู่ด้านบนกลางของ 2A5
    4. (2A8) Weapon Control Group (WCG) Unit
      1. สวิตช์ควบคุม "Test Set"
      2. สวิตช์ "Computer Mode"
        3. ๒๐๔.๒.๔ : (2A8) Weapon Control Group (WCG) Unit
        ประมวลผลข้อมูลทางเข้าและส่งข้อมูลออกให้ระบบย่อยต่างๆ
        ด้านขวาของตู้ ELX
        ๒๐๔.๒.๔ ก. : สวิตช์ควบคุม "Test Set"
        ในตําแหน่ง Enable จะยอมให้มีการทดสอบระบบแต่จะไม่ยอมให้มีการอาน ข้อมูลจาก ROM ในตําแหน่ง Disable จะเป็นการ Disable วงจรทดสอบ
        ด้านล่างซ้ายของ 2A8
        Enable/Disable
        ในตําแหน่ง Enable ยอมให้วงจรทดสอบทํางาน ในตําแหน่ง Disable ไม่ยอมให้ วงจรทดสอบ ทํางาน
        Disable
        ๒๐๔.๒.๔ ข. : สวิตช์ "Computer Mode"
        เปลี่ยนให้คอมพิวเตอร์อยู่ในตำแหน่ง Stop หรือ Run ไฟสีแดงติดแสดงว่าอยู่ใน ตำแหน่ง Run
        ด้านล่างขวาของ 2A8
        Stop/Run
        Run ยอมให้การทำงานเป็นไปอย่างปกติ Stop หยุดการทำงานของ คอมพิวเตอร์
        Run
    5. (2A15) Junction Box
      1. สวิตช์ "CB1"
      2. สวิตช์ "CB2"
        3. ๒๐๔.๒.๕ : (2A15) Junction Box
        เป็นที่ติดตั้ง CBS1, 2 (MODS 1-4,6) เป็นที่ติดตั้ง CB1 (MODS 11-14)
        ด้านขวาของตู้ ELX
        ๒๐๔.๒.๕ ก. : สวิตช์ "CB1"
        ควบคุมไฟ 440V. 60Hz 3 เฟส ไฟ EM ให้กับระบบ CIWS
        ภายใน 2A15
        ป้องกันการ Overload
        ตําแหน่ง ON (ผลักขึ้น)
        ๒๐๔.๒.๕ ข. : สวิตช์ "CB2"
        ควบคุมไฟ 440V. 60Hz 3 เฟส ไฟ ELX ให้กับระบบ CIWS
        อยู่ภายใน 2A15
        ป้องกันการ Overload
        ตำแหน่ง ON (ผลักขึ้น)
    6. (2A6A1) Power Supply and Control Group (PSCG) Panel
      1. สวิตช์ "CBS 1-9"
        2. ๒๐๔.๒.๖ : (2A6A1) Power Supply and Control Group (PSCG) Panel
        ควบคุมไฟการแจกจ่ายไฟทางออกของ PSCG
        ด้านซ้ายล่างของตู้ ELX
        ๒๐๔.๒.๖ ก. : สวิตช์ "CBS 1-9"
        ควบคุมการจ่ายไฟให้กับระบบย่อยต่างๆ
        ด้านหน้า 2A6A1
        ตําแหน่ง ON (ผลักขึ้น)
    7. (2A12A1) PSCG Panel
      1. สวิตช์ "CBS 1-9"
        2. ๒๐๔.๒.๗ : (2A12A1) PSCG Panel
        ควบคุมการแจกจ่ายไฟทางออกของ PSCG
        ด้านล่างขวาของตู้ ELX
        ๒๐๔.๒.๗ ก. : สวิตช์ "CBS 1-9"
        ควบคุมการจ่ายไฟ 440Vac
        ด้านหน้าของ 2A12A1
        ในตำแหน่ง ON (กดเข้า)
    8. (2A13) Liquid to Liquid Heat Exchanger
      1. M-1 Flow Meter
      2. V-1 Globe Valve
        3. ๒๐๔.๒.๘ : (2A13) Liquid to Liquid Heat Exchanger
        เป็นอุปกรณ์ระบายความร้อนจากสารหล่อเย็นไปยังน้ำทะเลแล้วระบายออกนอก ตัวเรือ
        ด้านซ้ายมือของตู้ ELX
        ๒๐๔.๒.๘ ก. : M-1 Flow Meter
        แสดงจํานวนอัตราการไหลของน้ําทะเล (20-25 gpm)
        อยู่ด้านล่าง Heat Exchange
        ๒๐๔.๒.๘ ข. : V-1 Globe Valve
        ควบคุมอัตราการไหลของน้ําทะเลที่ผ่าน Heat Exchanger
        ใต้ตู้ ELX
    9. (2A16) Suppressor Assembly Indicator
      10. ๒๐๔.๒.๙ : (2A16) Suppressor Assembly Indicator
      แสดงผลว่าไฟ 440V.ac 3 เฟส ที่จะจ่ายให้ระบบ CIWS พร้อมใช้งาน
      ด้านล่างขวาของตู้ ELX
      แผงสวิตช์บอร์ดในเรือ
  3. หลักการทํางาน
    1. ส่วนประกอบต่างๆ ทํางานร่วมกันในระบบอย่างไร
      2. ๒๐๔.๒.๙ : ส่วนประกอบต่างๆ ทํางานร่วมกันในระบบอย่างไร
      มีการจ่ายไฟสัญญาณคําสั่งควบคุมและป้องกันอุปกรณ์อิเล็กทรอนิกส์จาก สภาพแวดล้อมภายนอก
  4. ค่าทํางานปกติ ค่าสูงสุด ต่ำสุด ของการทํางาน
    5. อธิบายถึงค่าการทํางานปกติ ค่าสูงสุด ต่ำสุด ของอุปกรณ์ โดยใช้คําถาม ดังต่อไปนี้
    1. ค่าที่แสดงถึงการทํางานปกติ
    2. ขีดจํากัดสูงสุดที่อนุญาตให้ทํางานได้
    3. อ่านค่าได้จากที่ไหน
    4. ค่าที่จุดใดที่ตั้งไว้เพื่อใช้แจ้งเป็นสัญญาณเตือน
    คำตอบ: ก. ข. ค. ง.
    1. มิเตอร์ M -1 Flowmeter
      2. ๒๐๔.๔.๑ : มิเตอร์ M -1 Flowmeter
      ๒๐-๒๕ แกลลอนต่อนาที
      ๑๐-๒๕ แกลลอนต่อนาที
      อยู่ใต้ชุดระบายความร้อนของเหลวสู่ของเหลว (Liquid to Liquid Heat Exchanger)

  5. การเชื่อมต่อระบบ
    1. ถ้าขาดสิ่งดังต่อไปนี้จะมีผลกระทบต่อระบบอย่างไร
      1. ไม่มีน้ำหล่อเย็นจากภายนอก
      2. ไม่มีไฟฟ้าเรือจ่ายให้กับระบบ
      ๒๐๔.๕.๑ ก. : ไม่มีน้ำหล่อเย็นจากภายนอก
      เป็นสาเหตุทำให้อุปกรณ์อิเล็กทรอนิกส์ภายในร้อนเกินไป
      ๒๐๔.๕.๑ ข. : ไม่มีไฟฟ้าเรือจ่ายให้กับระบบ
      ELX Enclosure ไม่ทํางาน
  6. ข้อระมัดระวังอันตราย
    1. มีข้อระมัดระวังอันตรายเกี่ยวกับ Fire Integrity Key Switch
      2. ๒๐๔.๖.๑ : มีข้อระมัดระวังอันตรายเกี่ยวกับ Fire Integrity Key Switch
      ต้องแน่ใจว่าสวิทช์อยู่ในตำแหน่ง Open จนกว่าจะทำการยิง
