      10. ๒๐๒.๒.๔ : ภาคส่วนควบคุมขั้นการทํางาน "Mode Control Section"
      สําหรับเลือก Mode การทํางานต่างๆ ของระบบ CIW
      ด้านบนมุมซ้ายของบนตู้อุปกรณ์ LCP
      ๒๐๒.๒.๔ ก. : สวิตช์ปุ่มกดไฟแสดง "Battery Off"
      เมื่อกดปุ่มนี้ CIWS จะอยู่ในโหมดการทํางาน Battery Off ไฟที่สวิตช์จะติดสีขาว
      ขาว / -
      ๒๐๒.๒.๔ ข. : สวิตช์ปุ่มกดไฟแสดง "XMTR Coolant"
      ควบคุมการทํางานของ Transmitter Coolant Environment . ให้ทํางานไฟที่สวิตช์จะเปลี่ยนจากสีเหลืองเป็นสีเขียวก็ต่อเมื่อ ระบบ Coolant พร้อมใช้งาน
      เหลือง / เขียว
      ๒๐๒.๒.๔ ค. : สวิตช์ปุ่มกดไฟแสดง "Standby"
      เมื่อกดปุ่มนี้จะสั่งให้ระบบ CIWS อยู่ในโหมด Standby และไฟจะเปลี่ยนจากสี ขาวเป็นสีเขียวก็ต่อเมื่อเงื่อนไข Standby พร้อม และจะดับเมื่อเลือกโหมดการทํางานอื่นๆ
      ขาว / เขียว
      ๒๐๒.๒.๔ ง. : สวิตช์ปุ่มกดไฟแสดง "Stow Engaged/Stow Retract"
      ควบคุมเคลื่อนตัวเข้า - ออก ของ Stow Pin ถ้า Stow Pin อยู่ในตำแหน่ง Stow Engaged ไฟจะติดสีเหลือง ถ้าอยู่ในตำแหน่ง Retract ไฟจะติดสีเขียว
      เหลือง / เขียว
      ๒๐๒.๒.๔ จ. : สวิตช์ปุ่มกดไฟแสดง "Air Ready"
      เมื่อกดปุ่มนี้จะสั่งให้ระบบ CIWS อยู่ในโหมดการทำงาน Air Ready และไฟ แสดงสีขาวจะเปลี่ยนเป็นสีเขียวเมื่อ WCG คอมพิวเตอร์สั่งให้อยู่ในโหมด Air Ready และไฟจะดับเมื่อมีการเลือกการทำงาน Mode อื่นๆ
      ขาว / เขียว
      ๒๐๒.๒.๔ ฉ. : สวิตช์ปุ่มกดไฟแสดง "AAW Manual"
      เมื่อกดปุ่มนี้ WCG คอมพิวเตอร์จะสั่งให้ระบบ CIWS อยู่ในโหมดการทำงาน AAW Manual และไฟแสดงสีขาวจะเปลี่ยนเป็นสีเขียว และไฟจะดับเมื่อมีการเลือกโหมดการ ทำงานอื่นๆ
      ขาว / เขียว
      ๒๐๒.๒.๔ ช. : สวิตช์ปุ่มกดไฟแสดง "AAW Auto"
      เมื่อกดปุ่มนี้ WCG คอมพิวเตอร์จะสั่งให้ระบบ CIWS อยู่ในโหมดการทำงาน AAW Auto ไฟแสดงสีขาวจะเปลี่ยนเป็นสีเขียว และไฟจะดับเมื่อมีการเลือกโหมดการทำงานอื่น
      เปลี่ยนเป็นสวิตช์ที่มีฝาครอบ
      ขาว / เขียว
      ๒๐๒.๒.๔ ซ. : สวิตช์ปุ่มกดไฟแสดง "Surface"
      เมื่อกดปุ่มนี้ WCG คอมพิวเตอร์จะสั่งให้ระบบ CIWS อยู่ในโหมดการทำงาน Surface และไฟแสดงสีขาวจะเปลี่ยนเป็นสีเขียว ไฟจะดับเมื่อมีการเลือกโหมดการทำงาน อื่น ๆ ไม่มีใช้งานใน ทร.ไทย
      ขาว / เขียว
      ๒๐๒.๒.๔ ฌ. : สวิตช์ปุ่มกดไฟแสดง "Remote DESIG"
      เมื่อกด LCP จะต่อสัญญาน Synchro ที่เป็นข้อมูลเป้าหมายที่ได้รับการชี้เป้าส่งให้กับ Synchro ของแท่นยิงและส่งสัญญาณ Remote Designate ให้กับ WCG ที่สวิตช์จะมีไฟ แสดงสีขาวเมื่อกด จะเปลี่ยนจากสีขาวเป็นสีเขียวเมื่อ WCG สั่งให้ทำงานในโหมดนี้แล้ว
      ขาว / เขียว
