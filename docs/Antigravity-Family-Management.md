# 👨‍👩‍👧‍👦 Antigravity Family Quota Management Guide
คู่มือการบริหารจัดการโควตา Gemini Family (5 บัญชี) เพื่อการทำงานต่อเนื่อง 24 ชม.

---

## 🚀 1. การเตรียมการ (Setup Once)
ทำขั้นตอนข้างล่างนี้ "ครั้งเดียว" สำหรับสมาชิกแต่ละคน (รวม 5 คน)

### ขั้นตอนสร้าง "บ้านแยกส่วน" (Profiles)
1. กดคีย์ลัด **`Ctrl + Shift + P`** (Windows) หรือ `Cmd + Shift + P` (Mac)
2. พิมพ์คำว่า `Profile` แล้วเลือก **Profiles: New Profile...**
3. ตั้งชื่อตามสมาชิก (เช่น `User_01`, `User_02`) และเลือก Contents เป็น **None**
4. กดปุ่ม **Create** สีฟ้า

### ขั้นตอนผูกบัญชี (Sign In)
1. เมื่ออยู่ใน Profile ใหม่ ให้ไปที่เมนู **Account** (มุมซ้ายบนสุดในหน้า Settings `Ctrl + ,`)
2. หากมีบัญชีเก่าค้างอยู่ ให้กด **Sign Out** ก่อน
3. กด **Sign In** ด้วย Email ของสมาชิกคนนั้นๆ (ตรวจสอบว่าขึ้น **Google AI Pro**)

### ขั้นตอนดึงงานเข้า (Add Project)
1. ในหน้า Profile ใหม่นั้น ให้เลือกเมนู **File > Open Folder...**
2. เลือกโฟลเดอร์โปรเจกต์เดิม เพื่อให้ทุกคนแก้ไขไฟล์เดียวกัน

---

## ⚡ 2. คีย์ลัดสำหรับการใช้งานจริง (Daily Shortcuts)
ใช้คีย์ลัดเหล่านี้เพื่อความรวดเร็วในการสลับหน้าจอและบัญชี



| คำสั่ง | คีย์ลัด (Windows) | หน้าที่ |
| :--- | :--- | :--- |
| **สลับ Profile** | `Ctrl + Shift + P` > พิมพ์ `Switch Profile` | สลับบัญชีเมื่อโควตาคนปัจจุบันเต็ม |
| **สลับ Account** | `Ctrl + ,` > เลือก Account | เช็กสถานะ Plan หรือ Sign Out |
| **เรียก AI Agent** | `Ctrl + L` | เปิดหน้าต่างแชทกับ Gemini |
| **สั่ง AI เขียนโค้ด** | `Ctrl + I` | แก้ไขโค้ดในบรรทัดที่เลือกโดยตรง |

## 🛠️ 3. กลยุทธ์การทำงานร่วมกัน (Team Strategy)

### การส่งต่องาน (Handover)
เมื่อโควตาของสมาชิกคนหนึ่งหมด (Rate Limit Reached):
- **Step 1:** ให้คนปัจจุบันพิมพ์สรุปงานสั้นๆ ไว้ในไฟล์ `TODO.md` หรือท้ายไฟล์ `LOG.md`
- **Step 2:** กดสลับไป Profile ของสมาชิกคนถัดไป
- **Step 3:** ใน Profile ใหม่ ให้บอก AI ว่า: *"อ่านไฟล์ TODO.md และสานต่อจากจุดนั้นด้วยโควตาของคุณ"*

### การป้องกันการหลง Profile
- แนะนำให้ตั้ง **Color Theme** ของแต่ละ Profile ให้ต่างกัน เพื่อให้รู้ทันทีว่าใช้โควตาใครอยู่
- **วิธีเปลี่ยนสี:** `Ctrl + Shift + P` > พิมพ์ `Color Theme` > เลือกสีที่ชอบ

## ⚠️ ข้อควรระวัง
- **No API Key:** ไม่ต้องนำ API Key มาใส่ในหน้า Settings เพื่อป้องกันโควตาไหลรวมกัน
- **Git Commit:** ก่อนสลับ Profile แนะนำให้ Save หรือ Commit งานไว้เสมอ เพื่อป้องกัน Conflict
- **Independent Quota:** โควตาจะ Reset ทุกๆ 5 ชั่วโมง การมี 5 บัญชีจะทำให้คุณมี AI ใช้ได้เกือบตลอดเวลา
