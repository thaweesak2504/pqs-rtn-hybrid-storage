# Skill: Normalize Docs

Skill นี้ใช้เมื่อต้องจัดการไฟล์ข้อมูลดิบ (Raw Data) ประเภท CSV หรือ Markdown Table ที่ยังไม่สมบูรณ์



## 2. การตรวจสอบ (Verification)
- เปิดไฟล์ด้วย Text Editor
- ตรวจสอบบรรทัดท้ายไฟล์ว่าไม่มีการตกหล่น
- หากเป็น Markdown Table ให้จัด Format เป๊ะๆ ด้วย Prettier หรือ Extension
